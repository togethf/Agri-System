# TabBar 图标说明

本目录存放底部导航栏的图标资源。

## 所需图标

请使用以下图标或替换为您自己的图标：

1. **home.png / home-active.png** - 首页图标（灰色/绿色）
2. **camera.png / camera-active.png** - 拍照图标（灰色/绿色）
3. **consult.png / consult-active.png** - AI 问诊图标（灰色/绿色）

## 图标规格

- 尺寸：81px × 81px
- 格式：PNG（支持透明背景）
- 风格：简洁线性图标

## 临时方案

如果没有图标，可以暂时注释掉 `pages.json` 中的 `iconPath` 和 `selectedIconPath` 配置，tabBar 将只显示文字。
